Настройка не-проксируемых сайтов

Если вы хотите запретить проксирование определённых сайтов, используйте список исключений. Это может быть полезно, чтобы обходить прокси для некоторых ресурсов.

Пример настройки

Чтобы запретить проксирование сайтов ВКонтакте и RuTracker, добавьте следующие домены в список исключений:

vk.com rutracker.org

Настройка списка проксируемых сайтов

Список проксируемых сайтов позволяет использовать прокси только для выбранных ресурсов. Все остальные сайты не будут проходить через прокси.

Пример настройки

Чтобы использовать прокси только для сайта ChatGPT, добавьте следующий домен:

chatgpt.com
