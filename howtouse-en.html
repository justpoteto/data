Setting up non-proxied sites

If you want to prohibit proxying of certain sites, use the exception list. This can be useful to bypass proxies for some resources.

Example of setup

To disable proxying of Reddit and RuTracker sites, add the following domains to the exception list:

reddit.com rutracker.org

Setting up proxied sites

The list of proxied sites allows you to use a proxy only for selected resources. All other sites will not go through the proxy.

Example of setup

To use proxy only for ChatGPT site, add the following domain:

chatgpt.com
